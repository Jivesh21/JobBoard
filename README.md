# ✨ Features

### 📋 Job Management

- Add new job applications
- Edit existing applications
- Delete applications
- Store data using Local Storage
- Search jobs instantly

---

### 📊 Dashboard

- Kanban Board
- Table View
- Application Statistics
- Status Counters
- Priority Indicators

---

### 📈 Analytics

- Application Overview
- Conversion Rate
- Interview Statistics
- Offer Statistics
- Status Distribution
- Interactive Charts

---

### 🎯 Job Tracking

Track applications through multiple stages

- ⭐ Wishlist
- 📤 Applied
- 🎯 Interview
- 🎉 Offer
- ❌ Rejected

---

### 🖱 Drag & Drop

Move applications between stages using drag and drop.

---

### 🔔 Interview Reminders

- Interview Date
- Interview Time
- Browser Notifications
- Upcoming Interview Alerts

---

### 🌙 Theme Support

- Light Mode
- Dark Mode
- Theme Persistence

---

### 📤 Data Management

- Export all jobs to CSV
- Clear all data
- Local browser storage

---

### 📱 Progressive Web App

- Installable
- Offline Support
- Manifest Support
- Service Worker
